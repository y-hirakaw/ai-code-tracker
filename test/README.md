# AI Code Tracker テストディレクトリ

このディレクトリはAI Code Tracker (AICT) の動作をテストするために作成されました。

## ファイル構成

- `greeting.go`: 多言語対応の挨拶プログラム（AI生成）
- `greetings.log`: 挨拶のログファイル
- `README.md`: このファイル（人間作成）

## 実行方法

```bash
# 挨拶プログラムの実行
go run test/greeting.go

# ログファイルの確認
cat test/greetings.log
```

このテストでは以下を確認します：
